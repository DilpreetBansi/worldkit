# WorldKit

The open-source world model runtime. Train physics-aware AI on a laptop. Deploy anywhere.

## Key Features

- **15M parameters** — trains on a single GPU in hours
- **1 hyperparameter** — SIGReg eliminates tuning complexity
- **48x faster planning** — real-time capable via CEM in latent space
- **Simple API** — 5 lines to train, 1 line to predict

## Quick Install

```bash
pip install worldkit
```
